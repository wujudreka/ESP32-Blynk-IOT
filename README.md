# ESP32-Blynk-IOT
this is a simple code for you to turn on and off led built in of your ESP32 and display analog value through internet on your blynk IOT platform.

## what you need
1. Arduino IDE https://www.arduino.cc/en/software (tested on windows 11 version 1.8.19)
2. ESP32 Board library version 1.0.6 (ESP32 38 pin tested)
